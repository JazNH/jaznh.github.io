JUnit Testing

What is JUnit?

JUnit is a framework used for testing in Java

But what does that exactly mean? Well, lets say you created a method that takes a string and wants to make sure that that string ins't changed. You can use JUnit to make sure it's the same string from before. (Unless you changed the string then make sure its what you want it to be. But for this example lets just keep it simple)

Here are some very simple examples of how you can use JUnit when you are learning to test

AssertEquals
